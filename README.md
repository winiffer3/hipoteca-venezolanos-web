# Hipoteca Venezuela

Web de asesoría hipotecaria especializada en venezolanos residentes en España.

## Stack

- React 19.2.6 + Vite
- React Router DOM
- CSS Variables

## Instalación

npm install
npm run dev

## Estructura

src/
  pages/        # páginas de la web
  components/   # componentes reutilizables
docs/           # documentación y bocetos
public/images/  # imágenes y logo


## Páginas

| Ruta        | Página |
|-------------|--------|
| `/`         | Home |
| `/about`    | Nosotros |
| `/contacto` | Contacto |
| `/*`        | 404 |
